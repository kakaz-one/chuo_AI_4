## if文による条件分岐

### 処理の条件分岐 (1/3)

**条件分岐の例**

朝起きて外出前に降水確率を調べ、降水確率が50%以上の場合に傘を持ってから外出する。降水確率が50%を下回る場合は傘を持たずそのまま外出する。

→ 降水確率によって行う処理（行動）を分けている

> **プログラミングにおいて、ある条件が満たされているかどうかによって次に行う処理を変更する命令を条件分岐と呼ぶ**

```
処理開始
    ↓
降水確率を調べる
    ↓
降水確率50%以上？ ─YES→ 傘を持つ
    │                      ↓
    NO                  外出する
    │                      ↓
    └─────────────────→ 処理終了
```

### 処理の条件分岐 (2/3)

**ブール型とは**

ブール型（bool または boolean）は、真（True）と偽（False）という2値をとるデータ型である。

前ページの「降水確率50%以上かどうか」という条件式の場合：
- 降水確率が50%以上であれば、真と扱われる
- 降水確率が50%未満であれば、偽と扱われる

条件式の結果の真偽に合わせて、以降の処理が実行されるかどうかをコントロールする：
- 条件式の結果が真の場合、処理を実行する
- 条件式の結果が偽の場合、処理を実行しない

> **条件分岐を行う条件式の結果はブール型になる**

### 処理の条件分岐 (3/3)

**条件分岐の書き方**

> **Python で条件分岐を実装するには if文 を使用する**

```python
if 条件式:
    条件式がTrueのときに行う処理
```

- 処理内容を記述する前にインデント（4つの半角スペース）が必要※
- 条件式の最後にコロン（:）を記載

※ Python のバージョンによっては、4つ以外の半角スペースでも動作する

**使用例**

変数 `chance_of_rain`（降水確率）の値により処理が変わる：
- 0.5以上のときに、「傘が必要です」と表示される
- 0.5より小さいときには、なにも処理が行われない

```python
if chance_of_rain >= 0.5:
    print('傘が必要です')
```

### 複数条件式がある場合

> **複数条件分岐がある場合には、if文 を複数使用する**

```python
if 条件式1:
    条件式1がTrueのときに行う処理
if 条件式2:
    条件式2がTrueのときに行う処理
```

**使用例**

変数 `chance_of_rain`（降水確率）の値により処理が変わる：
- 0.5以上のときに、「傘が必要です」と表示される
- 0のときに、「雨の心配はありません」と表示される
- それ以外の場合には、なにも処理が行われない

```python
if chance_of_rain >= 0.5:
    print('傘が必要です')
if chance_of_rain == 0:
    print('雨の心配はありません')
```

### 条件分岐の中での条件分岐

> **条件分岐の中で、さらに条件分岐を行うことも出来る**

```python
if 条件式1:
    条件式1がTrueのときに行う処理
    if 条件式2:
        条件式2がTrueのときに行う処理
```

処理内容を記述する前にインデント（4つの半角スペース）が必要

**使用例**

変数 `chance_of_rain`（降水確率）と、`is_holiday`（休日かどうかのフラグ、休日:1、平日:0）の値によって処理が変わる：
- `is_holiday` の値が0、`chance_of_rain` の値が0.5以上のときに、「傘が必要です」と表示される
- それ以外の場合には、なにも処理が行われない

```python
if is_holiday == 0:
    if chance_of_rain >= 0.5:
        print('傘が必要です')
```

### elif文

> **elif文 を使用すると、elif文 より前の条件式を満たさなかった場合の条件分岐を行うことができる**

```python
if 条件式1:
    条件式1がTrueのときに行う処理
elif 条件式2:
    条件式1がFalseかつ、条件式2がTrueのときに行う処理
```

**使用例**

変数 `chance_of_rain`（降水確率）の値により処理が変わる：
- 0.5以上のときに、「傘が必要です」と表示される
- 0.5より小さい、かつ0.3以上のときに「折り畳み傘が必要です」と表示される
- それ以外の場合には、なにも処理が行われない

```python
if chance_of_rain >= 0.5:
    print('傘が必要です')
elif chance_of_rain >= 0.3:
    print('折り畳み傘が必要です')
```

※ `if 0.3 <= chance_of_rain < 0.5:` と等価だが、elifの方が簡潔に書ける

### else文

> **else文 を使用すると、else文 より前の条件式を満たさなかった場合の処理を行うことができる**

```python
if 条件式1:
    条件式1がTrueのときに行う処理
else:
    条件式1がFalseのときに行う処理
```

※ else文では条件式を書かない

**使用例**

変数 `chance_of_rain`（降水確率）の値により処理が変わる：
- 0.5以上のときに、「傘が必要です」と表示される
- 0.5より小さい、かつ0.3以上のときに「折り畳み傘が必要です」と表示される
- それ以外の場合には、「お知らせはありません」と表示される

```python
if chance_of_rain >= 0.5:
    print('傘が必要です')
elif chance_of_rain >= 0.3:
    print('折り畳み傘が必要です')
else:
    print('お知らせはありません')
```

### 比較演算 (1/4)

比較演算の戻り値はTrue、Falseのブール型である。

> **条件式には結果が真偽値となる式であれば何でも記述できるが、比較演算子を含む式を記述することが多い**

| 演算子 | 意味 |
|--------|------|
| `a == b` | aとbの値が等しい |
| `a != b` | aとbの値が等しくない |
| `a > b` | aの値がbの値より大きい |
| `a < b` | aの値はbの値より小さい |
| `a >= b` | aの値はbの値より大きいか等しい |
| `a <= b` | aの値はbの値より小さいか等しい |
| `a in b` | aはbを包含する |
| `a is b` | aとbは同一オブジェクトである |
| `a is not b` | aとbは同一オブジェクトではない |

### 比較演算 (2/4)

> **==演算子を条件式に用いることで、二値が一致しているかどうかを判定できる**

**使用例：==演算子を用いて二つの文字列が同値か判断する**

```python
if sentence == 'I live in Tokyo.':
    print('東京に住んでいます')
```

- `sentence` の値が `'I live in Tokyo.'` のときに、「東京に住んでいます」と表示される
- それ以外の場合には、なにも処理が行われない

**使用例：==演算子を用いて二つのリストが同値か判断する**

```python
if prefectures == ['Tokyo', 'Nagoya', 'Osaka']:
    print('同じリストである')
```

- `prefectures` が `['Tokyo', 'Nagoya', 'Osaka']` のときに、「同じリストである」と表示される
- それ以外の場合には、なにも処理が行われない

### 比較演算 (3/4)

> **不等号を条件式に用いることで、二値の大小を判定できる**

**使用例：不等号を用いて二値の大小を判定する**

```python
if num > 5:  # numが5より大きいか判定
    print('"num"は5よりも大きいです')
```

- `num` が5よりも大きいときに、「"num"は5よりも大きいです」と表示される
- それ以外の場合には、なにも処理が行われない

**使用例：不等号を条件式に複数用いて大小を判定する**

```python
if 10 <= age < 20:
    print('10代です')
elif 20 <= age < 30:
    print('20代です')
```

- `age` の値が10以上で、20未満のときに、「10代です」と表示される
- `age` の値が10以上20未満でないかつ、20以上30未満のときに、「20代です」と表示される
- それ以外の場合には、なにも処理が行われない

### 比較演算 (4/4)

> **in 演算子を条件式に用いることで、文字列やリストなどに特定の要素が含まれているかどうかを判定できる**

**使用例：in 演算子を用いて文字列に部分文字列が含まれるかどうかを判定する**

```python
sentence = 'I live in Tokyo.'
if 'Tokyo' in sentence:
    print('文字列に"Tokyo"が含まれる')
```

- `sentence`の中に`'Tokyo'`が含まれているとき、「文字列に"Tokyo"が含まれる」と表示される
- それ以外の場合には、なにも処理が行われない

**使用例：in 演算子を用いてリストに要素が含まれているかどうかを判定する**

```python
prefectures = ['Tokyo', 'Nagoya', 'Osaka']
if 'Osaka' in prefectures:
    print('リストに"Osaka"が含まれる')
```

- `prefectures`に`'Osaka'`が含まれているときに、「リストに"Osaka"が含まれる」と表示される
- それ以外の場合には、なにも処理が行われない

### ブール演算 (1/4)

> **比較演算は and、or および not のブール演算子を組み合わせることができる**

**ブール演算子について**
- ブール演算子は `==` や `<` などの比較演算子より優先順位が低い
- ブール演算子の優先順位は、not → and → or の順である

| 演算 | 説明 | 図解 |
|------|------|------|
| `A and B` | 論理積 | A ∩ B |
| `A or B` | 論理和 | A ∪ B |
| `not A` | 否定 | Aの補集合 |

### ブール演算 (2/4)

> **「A and B and C」の場合、A、B、C がすべて真（True）の場合はTrue、1つでも偽（False）がある場合は False となる**

**and を用いた条件式の書き方**

```python
if 条件式1 and 条件式2:
    条件式1と条件式2がともにTrueのときに行う処理
```

**使用例**

```python
a = 1
b = 2
if a == 1 and b == 2:
    print('aの値は1でbの値は2です')
```

- aの値が1かつbの値が2のときに、「aの値は1でbの値は2です」と表示される
- aの値が1でないまたはbの値が2でないときには、なにも処理が行われない

### ブール演算 (3/4)

> **「A or B or C」の場合、A、B、C のうち1つでも真（True）の場合はTrue、すべて偽（False）である場合は False となる**

**or を用いた条件式の書き方**

```python
if 条件式1 or 条件式2:
    条件式1または条件式2がTrueのときに行う処理
```

**使用例**

```python
height = 140
if height < 150 or height >= 180:
    print('身長は150センチ未満または180センチ以上です')
```

- `height` の値が150未満または180以上のときに、「身長は150センチ未満または180センチ以上です」と表示される
- `height` の値が150以上かつ180未満のときには、なにも処理が行われない

### ブール演算 (4/4)

> **ブール演算子は複数組み合わせることができる**

**優先的に評価したい部分は () をつけて対処する**

**使用例：x が「10未満または20以上」かつ「偶数」であるかどうかを判定する**

```python
if (x < 10 or x >= 20) and x % 2 == 0:
    print('OK')
else:
    print('NG')
```

- 優先的に評価したい箇所に () を付けて条件式を記述する
- 上記の例で () をつけずに条件式を書くと、or より and が優先されて「20以上かつ偶数」または「10未満」という条件式になってしまい、想定している評価ができなくなる

## for文による繰り返し

### 繰り返し処理 (1/2)

> **決まった回数や条件を満たしている間に同じ処理を繰り返し行うことができる**

繰り返し処理を行うために for文と while文が利用できる。

**for文**
- 決まった回数繰り返し処理を行う
- 要素をすべて取り出す、または指定回数実行したらループを抜ける

**while文**
- 条件式を記述し、条件が真の間処理を繰り返し行う
- 無限ループを記述できる

### 繰り返し処理 (2/2)

> **繰り返し処理を使うと、同じ処理を行うコードを簡潔に書くことができる**

**繰り返し処理を使用しない書き方の例**

```python
print('佐藤さん、おはようございます。')
print('鈴木さん、おはようございます。')
print('小林さん、おはようございます。')
print('加藤さん、おはようございます。')
# 同じようなコードを10回書いている
```

**for文を用いた繰り返し処理の例**

```python
name_list = ['佐藤', '鈴木', '高橋', '田中', '伊藤', 
             '渡辺', '山本', '中村', '小林', '加藤']
for name in name_list:
    print(name + 'さん、おはようございます。')
```

どちらも実行結果は同じである。繰り返し処理を用いれば、同じ処理内容を簡潔に書くことができる。

### シーケンスに対する繰り返し処理

> **シーケンスに対して同じ処理を繰り返し行いたいときは for文 を使う**

リストや文字列のような複数の要素が順番に並んだデータをシーケンスと呼ぶ。

**for文 の書き方**

```python
for 任意の変数 in シーケンス:
    ループ内で実行する処理
```

- 処理内容を記述する前にインデント（4つの半角スペース）が必要※
- 最後にコロン（:）を記載

※ Python のバージョンによっては、4つ以外の半角スペースでも動作する

**使用例：リスト内の文字列を順に出力したい場合**

```python
words = ['cat', 'lion', 'albert']
for word in words:
    print(word)
```

**実行結果**
```
cat
lion
albert
```

### 決まった回数の繰り返し処理 (1/2)

> **指定した回数だけ繰り返したいとき、組み込み関数の range()関数 を使用する**

**range() の書き方**

```python
range([始まりの数値,] 終わりの数値 [, 増加する量])
```

- 上記の3つの引数のうち、「始まりの数値」と「増加する量」は省略可能
- `range()` に数値を一つ渡すと「終わりの数値」となり、0から「終わりの数値-1」までの値が格納されたシーケンスオブジェクトが生成される

**使用例**

```python
range(10)           # ① range(0, 10) - 0から始まり9で終わる連番
range(1, 20)        # ② range(1, 20) - 1から始まり19で終わる連番
list(range(2, 10, 3))  # ③ [2, 5, 8] - 2から9まで3つ間隔でリストを生成
```

### 決まった回数の繰り返し処理 (2/2)

> **指定した回数だけ繰り返したいとき、組み込み関数の range()関数 を使用する**

**range() を用いた繰り返し処理の例：5回ループを実行し、0～4までの整数を表示する**

```python
for count in range(5):
    print(count)
```

**実行結果**
```
0
1
2
3
4
```

※ `range()`に数値を1つ渡した場合、0から始まることに注意

### 繰り返し処理の中での条件分岐 (1/4)

> **for文 によるループを中断するには break文 を使用する**

**使用例：break文 を使用して、forループを中断する**

break文 はif文内で使用するのが一般的である。

```python
for i in range(5):
    print(i)
    if i == 2:
        break
```

**実行結果**
```
0
1
2
```

iが2となったとき、ループを抜けて処理終了

### 繰り返し処理の中での条件分岐 (2/4)

> **複数ループがある場合、break文 は最も内側のループを中断する**

**使用例**

```python
for i in range(1, 10):
    for j in range(1, 10):
        if i * j > 10:
            print(i, j)
            break
```

**実行結果**
```
2 6
3 4
4 3
5 3
6 2
7 2
8 2
9 2
```

`i * j > 10` となったとき、内側のループのみ抜ける

### 繰り返し処理の中での条件分岐 (3/4)

> **for文 によるループは else節 を持つことができる**

for文によるループでは、シーケンスのすべての要素を使い切ってループが終了した場合に実行される。

**for-else文 の書き方**

```python
for 任意の変数 in シーケンス:
    ループ内で実行する処理
else:
    シーケンスの要素を使い切った後に実行される処理
```

※ 行頭の位置はforと同じ位置

**使用例**

```python
for i in range(3):
    print(i)
else:
    print('Done')
```

**実行結果**
```
0
1
2
Done
```

### 繰り返し処理の中での条件分岐 (4/4)

> **for文 によるループで break文 によってループが中断される場合、else節内の処理は実行されない**

**使用例**

```python
for i in range(5):
    print(i)
    if i == 3:
        break
else:
    print('Done')
```

**実行結果**
```
0
1
2
3
```

※ `'Done'` は表示されない

## （参考）if文による条件分岐 応用編

### 比較演算 応用編 (1/3)

Pythonでは数値や文字列、リストはすべてオブジェクトである（第2回の授業を参照）。

> **オブジェクトが同じかどうかを判定するには is演算子 を使用する**

**is と == の違い**

| 演算 | 結果 |
|------|------|
| `is` | オブジェクトが同じであるかどうかを判定する比較演算子 |
| `==` | オブジェクトの値が等しいかどうかを判定する比較演算子 |

**使用例**

```python
a = [1, 2, 3]
b = a
c = [1, 2, 3]
print(a is b)  # True - aとbは同一オブジェクトと判定されている
print(c is a)  # False - cとaは同一オブジェクトと判定されていない
```

### 比較演算 応用編 (2/3)

> **is演算子 はオブジェクトIDを比較している**

オブジェクトのidは、`id()`関数で調べることができる（第2回の授業を参照）。

**id関数の使い方**

```python
id(オブジェクト)
```

**使用例**

```python
a = [1, 2, 3]
b = a
c = [1, 2, 3]
print(id(a))  # 2628256365896
print(id(b))  # 2628256365896 - aとbは同じオブジェクトID
print(id(c))  # 2628256503432
```

※ 環境によってオブジェクトIDの値は異なる

### 比較演算 応用編 (3/3)

> **オブジェクトの値が None であるか判定する場合は、is演算子 を用いる**

**None とは**

「何もない」、「空」を意味するオブジェクト。プログラミング言語によってはnullと呼ばれることもある。

**使用例**

```python
x = None
if x is None:
    print('xはNoneである')
```

**実行結果**
```
xはNoneである
```

- 条件式を「`x == None`」としても結果は同じだが、`==`演算子 はオブジェクトの同値性を判定するのに対し、`is`演算子 はオブジェクトの同一性を判定する
- Noneの比較においては、None が Python のメモリ上に唯一存在するオブジェクトであることから `is`演算子 で判定するのが正しい

### ブール演算 応用編 (1/3)

> **and と or は引数が左から右へと順に評価され、結果が確定した時点で評価が止まる**

and と or は短絡（short-circuit）演算子と呼ばれる。

| 演算 | 結果 |
|------|------|
| `x or y` | x が偽なら y、そうでなければ x |
| `x and y` | x が偽なら x、そうでなければ y |

**使用例**

```python
a = 1
b = 2
print(a > 1 or b > 1)   # True
print(a > 0 or b > 1)   # True
print(a > 1 and b > 1)  # False
print(a > 0 and b > 1)  # True
```

### ブール演算 応用編 (2/3)

> **and、or の戻り値はブール型以外にすることも可能であり、真偽に応じて左辺または右辺の値を返す**

and と or は短絡（short-circuit）演算子と呼ばれる。

**使用例**

```python
a = 1
print(a > 1 or 'success')   # success - xが偽なので評価が進んで'success'を返す
print(a > 0 or 'success')   # True - xが真なので評価は終了し、orの後ろは考慮しない
print(a > 1 and 'success')  # False - xが偽なので評価は終了し、andの後ろは考慮しない
print(a > 0 and 'success')  # success - xが真なので評価が進んで'success'を返す
```

### ブール演算 応用編 (3/3)

> **notは否定とも呼ばれるブール演算子である**

`not x` の場合、x が True ならば False に反転し、x が False ならば True に反転する。

| 演算 | 結果 |
|------|------|
| `not x` | xが偽ならTrue、そうでなければFalse |

**使用例**

```python
a = 1
print(a > 1)      # False
print(not a > 1)  # True
print(a > 0)      # True
print(not a > 0)  # False
```

## （参考）while文による繰り返し

### while文を使った繰り返し処理

> **while文を用いると条件式が真である間、実行を繰り返す処理を記述できる**

**while文の書き方**

```python
while 条件式:
    条件式がTrueのときに行う処理
```

- 処理内容を記述する前にインデント（4つの半角スペース）が必要※
- 条件式の最後にコロン（:）を記載

※ Python のバージョンによっては、4つ以外の半角スペースでも動作する

**使用例：a が10未満の場合 a の値を出力し、a に2をかける処理を繰り返す**

```python
a = 2
while a < 10:
    print(a)
    a *= 2
```

**実行結果**
```
2
4
8
```

### while文の中での条件分岐 (1/2)

> **while文の条件式が常に真だと、そのままでは無限ループになる**

常に真は一般的に `while True:` と記述する。

**無限ループを防ぐ書き方**

```python
while True:
    条件式がTrueのときに行う処理
    if 条件式:
        break
```

※ 必ずbreakとセットで使い、無限に処理が実行されないように注意する

**使用例**

```python
num = 0
while True:
    if num == 5:
        print('numの値が5になりました')
        break
    num = num + 1
```

numの値が5になったときに「numの値が5になりました」と表示されループを抜ける

### while文の中での条件分岐 (2/2)

> **while文 によるループは else節 を持つことができる**

while文によるループでは条件式が偽になった場合に実行される。

**while-else文の書き方**

```python
while 条件式:
    条件式が True のときに行う処理
else:
    条件式が False になった後に実行される処理
```

※ 行頭の位置は while と同じ位置

**使用例**

```python
i = 1
while i < 5:
    i += 1
else:
    print('iが5以上になりました')
```

**実行結果**
```
iが5以上になりました
```

### 繰り返し処理の中での条件分岐 応用編

> **for文 によるループ中で繰り返し処理を短絡するには continue文 を使用する**

※ while文によるループでも continue文 を使用することができる

**使用例**

```python
for n in range(2, 10):
    if n % 2 == 0:
        print(n, 'is an even number')
        continue
    print(n, 'is an odd number')
```

```
7 is an odd number

```

- `continue` が実行されると、以降の処理がスキップされてループの先頭に戻る
- `continue` が実行されなかったときに下の `print` が実行される

### ループのテクニック (1/3)

> **繰り返し処理（ループ）において、繰り返し可能なオブジェクトをイテラブルオブジェクトと呼ぶ**

**イテラブルオブジェクト**
- イテラブル（iterable）とは、「反復可能な、繰り返し可能な」という意味
- for文のin部分に書き込めるオブジェクト
- 例：range関数で生成されたオブジェクトやシーケンス型のオブジェクト

**使用例**

```python
li = ['A', 'B', 'C']
for i in li:  # liはリストなのでイテラブルオブジェクト
    print(i)
```

**実行結果**
```
A
B
C
```

### ループのテクニック (2/3)

> **繰り返し処理のイテラブルオブジェクトとしてenumerate()関数を使うと、要素をインデックスと同時に取得することができる**

**繰り返し処理時にenumerate()関数を使う場合の書き方**

```python
for 任意の変数1, 任意の変数2 in enumerate(イテラブルオブジェクト):
    繰り返し行いたい処理
```

処理内容を記述する前にインデント（4つの半角スペース）が必要

**使用例**

```python
for i, item in enumerate(['A', 'B', 'C']):
    print(i, item)
```

**実行結果**
```
0 A
1 B
2 C
```

※ `item` には 'A'、'B'、'C' が順に入る

### ループのテクニック (3/3)

> **繰り返し処理のイテラブルオブジェクトとしてzip()関数を使うと、複数の要素を持つデータ型を同時にループすることができる**

**繰り返し処理時にzip()関数を使う場合の書き方**

```python
for 任意の変数1, 任意の変数2 in zip(イテラブルオブジェクト1, イテラブルオブジェクト2):
    繰り返し行いたい処理
```

処理内容を記述する前にインデント（4つの半角スペース）が必要

**使用例**

```python
li = ['a', 'b', 'c']
tu = ('A', 'B', 'C')
for i, t in zip(li, tu):
    print(i, t)
```

**実行結果**
```
a A
b B
c C
```

- リスト型`li`の要素'a'、'b'、'c'を変数`i`に定義する処理を繰り返す
- タプル型`tu`の要素'A'、'B'、'C'を変数`t`に定義する処理を繰り返す

### 内包表記

> **リスト・セット・ディクショナリにおいて、for文などによるシンプルな処理をスマートに記述できる記法として内包表記がある**

**内包表記のメリット**
- 簡潔にコードを書ける
- 処理速度が速い

**内包表記を使わない書き方**

```python
リスト/セット/ディクショナリの初期値の変数
for 任意の変数 in 複数の要素を持つオブジェクト:
    要素について行う処理
```

**内包表記を使った書き方**

```python
変数 = [要素について行う処理 for 任意の変数 in 複数の要素を持つオブジェクト]
```

※ リストの場合は `[]`、セットもしくはディクショナリの場合は `{}` で囲む

### リスト型の内包表記

> **for文を使ってリストを生成する際に、内包表記を用いると簡潔に記載できる場合がある**

**内包表記を使わない場合**

```python
li = []
for x in range(5):
    li.append(x**2)
li
```

**内包表記を使う場合**

```python
li = [x**2 for x in range(5)]
li
```

**実行結果**
```
[0, 1, 4, 9, 16]
```
